		/*
		변수 선언 방법
		방법1
		var 변수명 = 값;
		var 변수명;

		방법2
		let 변수명 = 값;
		let 변수명;

		변수
		- 값을 저장하는 공간임
		- 값을 여러번바꿀수있음 말그대로 변수니깐. 
		- 예
		  -1000
			-볼펜값 : 1000 (볼펜값이 변수임. )

		var와 let의 차이
		 - var는 호이스팅이 일어나지만, let은 일어나지 않음.
		 - var는 동일한 변수를 중복선언해도 동작(체크를 안해줌), let은 예외 발생(체크를해줌)
		 - let은 사용을 추천 


		호이스팅
		 -페이지 로딩 시 변수나 함수가 위로 올려지는 동작
		 -var로 선언된 변수, 함수는 호이스팅이 적용.
		*/

		 

		let age = 20;

		/*
		변수명 작성 관례(카멜 표기법)
    -지키지 않아도 문법적으로는 상관이없음(코드상으로 문제는 없으나 나중에 확인하기가 힘듬.그래서 지키는게 좋음.)		
		-변수명은 의미있게 작성
		  - 잘못된 예시: a(사과의 갯수)
			- 잘된예 : appleCount
		 - 카멜표기법
		  -두단어 이상으로 된 변수명의 두번째 단어부터 첫글자를 대문자로 씀. (예시 appleCount	, appleCountHouse 이런식으로 결합어에서 두번째 단어부터는 대문자사용임.) 

		변수명(상수-다만 상수는 대문자로만) 작성 규칙

		-변수명은 숫자로 시작할 수 없음	(no.1은 되지만 1.no는 안됨.)
		-변수명의 특수문자는 "_"와 "$"만 (언더바와 달러만 가능)
		-예약어(키워드) 사용불가
		 -예약어 : 기능이 있는 단어로 const, let, class등등 단어에 기능이 있는것들은 사용하면 안됨.
		-변수명은 대소문자를 구별함
		 -num와 NUM는 다른변수 (소문자와 대문자를 별개로 취급함)
